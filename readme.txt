== Description ==

Chat-In es una solucion integral al momento de obtener contactos de tu página. Integra todo lo necesario para convertir tus clientes en ventas aseguradas, desde un chatbot 24/7 hasta un formulario de contacto. Toda la informacion de tus clientes se pueden ver desde un panel de control que integra toda la informacion necesaria.

Nuestro servicio está basado en www.chat-in.net, desde ahi podras registrarte gratuitamente, obtener una cuenta demo y comenzar a probar tanto el chatbot como el diseño de formularios.


== Installation ==

1. Subir `chat-in.php` a la carpeta `/wp-content/plugins/` 
2. Activar el plugin desde el mismo panel de Wordpress
3. Registrarse en www.chat-in.net, agregar un sitio web y desde el dashboard obtendras todos los datos necesarios para la conexion con Wordpress.
4. Ir a "Ajustes" -> "Chat-In" Y configurar el numero de empresa y formulario asignado en www.chat-in.net
5. Poner el formulario de contacto escribiendo [chatin-contact-form] en la página del sitio que quieras
